# 历史粘接 - 剪贴板历史管理器

Windows 剪贴板历史管理工具，后台静默记录所有复制过的文字和图片，支持搜索、置顶、一键复用。

## 功能特性

- **自动记录** — 后台静默监控剪贴板，文字和图片复制自动入库
- **智能去重** — 相同内容不重复记录，避免冗余
- **分类筛选** — 全部 / 文字 / 图片 三个标签切换
- **快速搜索** — 关键词搜索剪贴板历史，支持中文
- **一键复用** — 点击卡片直接复制回剪贴板
- **置顶/删除** — 常用内容置顶到顶部，不需要的随手删除
- **图片保存** — 历史中的图片可另存到本地
- **自动清理** — 支持 1/3/5 天自动清理过期记录
- **窗口透明** — 25%~100% 可调节透明度，滑块实时调整
- **窗口置顶** — 一键切换窗口置顶，始终悬浮
- **系统托盘** — 最小化到系统托盘，后台静默运行
- **全局热键** — `Ctrl+Shift+V` 随时随地呼出窗口
- **开机自启** — 可选开机自动启动（Windows 注册表）
- **单实例运行** — 重复启动自动唤醒已有窗口

## 界面预览

> 运行后按 `Ctrl+Shift+V` 呼出主界面

- **静默模式**：只显示内容卡片，界面干净简洁
- **搜索面板**：点击 🔍 展开搜索框 + 透明度 + 置顶控制

## 技术栈

- **语言**：Python 3.14
- **GUI**：tkinter（Python 内置，无需额外安装框架）
- **数据库**：SQLite（内置 `sqlite3`）
- **图像处理**：Pillow（`PIL`）
- **系统托盘**：pystray
- **全局热键**：keyboard
- **剪贴板访问**：pyperclip + ctypes + PIL.ImageGrab

## 安装与运行（小白友好版）

### 第一步：下载代码

点击页面右上角绿色 **Code** 按钮 → **Download ZIP**，解压到任意文件夹。

### 第二步：安装 Python

如果还没装 Python，去 [python.org](https://www.python.org/downloads/) 下载安装。

> **安装时务必勾选** `Add Python to PATH`（添加到系统环境变量）

### 第三步：安装依赖

打开 **命令提示符**（Win+R 输入 `cmd` 回车），粘贴以下命令：

```bash
pip install pillow pyperclip pystray keyboard
```

### 第四步：启动

在解压的文件夹中双击 `启动历史粘接.lnk`，应用即以后台托盘模式运行。

> 按 `Ctrl+Shift+V` 呼出窗口，开始使用。

### 创建桌面快捷方式（可选）

右键 `create_shortcut.ps1` → **使用 PowerShell 运行**，桌面即生成快捷方式。

## 数据存储

所有数据存储在用户 AppData 目录下：

```
%APPDATA%/clipboard-manager/
├── clipboard.db            # SQLite 数据库
├── images/                 # 图片文件存储
├── app.log                 # 运行日志
└── show.signal             # 单实例唤醒信号文件
```
